Cyberpunk Tactical Analyst: FPGA-Accelerated Game of Life

Welcome to the Tactical Analyst terminal. This project is a full-stack, hardware-accelerated implementation of Conway's Game of Life, featuring a custom Verilog processor deployed on a Xilinx PYNQ FPGA. It is augmented with Google's Gemini 2.5 Flash AI, which acts as a real-time tactical commander to analyze board states and recommend optimal cell placements.
⚡ The Tech Stack

    Hardware: Xilinx Zynq FPGA (PYNQ-Z2)

    Hardware Description: Verilog (Custom AXI-Lite IP)

    Backend: Python (Flask)

    Frontend: HTML5, CSS3 (Cyberpunk UI), Vanilla JavaScript

    AI Integration: Google Gemini 2.5 Flash API (Browser-side)

📂 Repository Structure

Based on the project architecture, the source code is organized as follows:

    /verilog_src: Contains the custom Verilog HDL code for the Game of Life processor. Includes the 16x16 grid logic, neighbor calculation combinational logic, and the edge-detected state machine for precise 1-FPS frame stepping.

    /Output_FPGA: Contains the synthesized .bit (bitstream) and .hwh (hardware handoff) files generated by Xilinx Vivado. These are the physical logic gates ready to be flashed to the board.

    /Frontend: Contains the Python Flask backend (app.py) and the templates/index.html file that drives the interactive, glowing Cyberpunk web interface.
